## What to Update

When rebranding, the following elements need to be changed:

### 1. Brand Identity
- Company name in all locations
- Tagline/slogan
- Logo (favicon and inline SVG)
- Social media URLs

### 2. File Locations

**index.html:**
- Meta description (line ~6)
- Page title (line ~7)
- Favicon SVG (line ~12)
- Logo text (nav and footer)
- Alt text on images
- Email addresses
- Social media links
- Copyright text
- Footer tagline

**styles.css:**
- Header comment (lines 1-5)

**script.js:**
- Header comment (lines 1-4)

### 3. Content Updates
- Hero headline and subtitle
- About section narrative
- Vision/Process section text
- Contact section CTA

## Instructions

To rebrand, please provide:
1. **New company name**
2. **New tagline** (primary and secondary if applicable)
3. **New email domain** (e.g., hello@newdomain.com)
4. **New social media handles** (LinkedIn, GitHub, Twitter)
